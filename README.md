# 🔥 Heat Trail Animation

This is a fun and interactive **heat trail animation** project created using **HTML**, **CSS**, and **JavaScript**. It creates a cool floating image trail (like heat sparks or flame icons) that follows your mouse cursor as you move it around the screen.

---

## ✨ Features

- 🔥 Mouse trail effect using flame icon
- 🎨 Animated movement and color transitions
- ⏱️ Particles auto-remove after a few seconds
- 🖱️ Fully interactive and responsive

---

## 📁 File Structure
  - 📁 heat-trail-animation/
  - ├── index.html # Main HTML file
  - ├── style.css # CSS styles for the heat trail
  - ├── index.js # JavaScript to generate trail animation
  - └── README.md # Project documentation



---

## 🚀 How to Run

1. Download or clone the repository.
2. Make sure `index.html`, `style.css`, and `index.js` are in the same folder.
3. Open `index.html` in your browser.
4. Move your mouse around and enjoy the animation trail!

---

## 🛠️ Technologies Used

- **HTML5** – Page structure
- **CSS3** – Styling and animations
- **JavaScript** – Dynamic element creation and behavior

---



---

## ⚠️ Notes

- You can change the icon in the CSS `span` selector using any other URL.
- The current animation uses a 5-second delay before removing the trail elements.
- To make the trail smoother or faster, adjust the `setTimeout()` and `@keyframes` animation settings.
